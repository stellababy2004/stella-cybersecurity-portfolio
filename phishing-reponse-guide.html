The first 10 minutes after detecting a phishing attack are crucial for preventing data loss and system compromise.

1 DO NOT PANIC - Stay calm and follow this protocol
2 Isolate the affected system immediately
  • Disconnect from network (unplug Ethernet cable)
  • Disable Wi-Fi if wireless connection
  • Do NOT shut down the computer (preserves evidence)
3 Alert the Security Team
  • Call SOC immediately: [INSERT SOC PHONE NUMBER]
  • Send urgent email to: security@company.com
  • Use incident response hotline if available
